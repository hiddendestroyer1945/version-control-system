# Version Control System
*   **Description**: To explain what is version control system and features of that.
*   **Version Control System**: It is a system that records changes to a file or set of files over time so that you can recall specific versions later. It's used for managing the project files and collaborating with others.
*   **Version Control System Features**:
 
    *   **History Tracking**: It is a system that records changes to a file or set of files over time so that you can recall specific versions later. It's used for project's specific activities recording and recalling.
    *   **Collaboration**: It used for remotely worked many persons collaborated on the same project. It's benefits are remote working persons working together, coordinating with each other and analyzing that persons activities.
    *   **Backup**: It used for keep the project files safe and secure. When needed to restore the project files.
    *   **Multiple Features**: It used for working safely and nonconflictingly in different project or same project's different components on same time.
    *   **Tracking Status**: It used for tracking the last changes and activities made to a specific project when working on it after a long time.
    *   **Open Source Project Using**: It used for doing the open source projects, and working on this projects.

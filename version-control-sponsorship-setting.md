# Version Control Sponsorship Setting

*   **Description**: To explain about the version control setting, profile sponsorship setting process and remote repository sponsorship setting.

*   **Version Control Sponsorship Setting**: Version control sponsorship settings are configured to accept donations from other user accounts on that specific Git server to the project's remote repository.

*   **Profile Sponsorship Setting**: To set up sponsorships for the project's remote repository to receive donations from other user accounts on that specific Git server, a Stripe Connect account is created on the Sponsors page of that user account, including the user account's personal information, bank information, and PAN card information.Then, after verification of that Stripe Connect account, the necessary sponsor settings are configured on that user account's Sponsors Dashboard page.

-  To set up sponsorships for the project's remote repository to accept contributions from other user accounts on that particular Git server, an account is created on any payment gateway platform that can connect to that Git server on the Sponsors page of that user account, which includes the user account's personal information, bank details, and PAN card information. Then, after verification of that Payment gateway Connect account, the necessary sponsor settings are configured on that user account's Sponsors Dashboard page.

*   **Remote Repository Sponsorship Setting**: To set up sponsorships for a project's remote repository to accept contributions from other user accounts on the Git server, select the Sponsorships option in the project's remote repository settings, click the Setup Sponsor button, and enter the Stripe Connect account's user account information against the name of the specified Git server in the resulting file.

-  To set up sponsorships for a project's remote repository to accept contributions from other user accounts on the Git server, select the Sponsorships option in the project's remote repository settings and click the Setup Sponsor button. In the resulting file, enter the user account information for the payment gateway platform against the name of the specified payment gateway platform.
